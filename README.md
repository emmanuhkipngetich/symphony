# Reja reja app automation
#### This is an automation script for reja reja app. 
#### By Polycarp Kavoo.
## Description
The project automates login, logout and add to cart.
Recorded on testproject.io and set up on pycharm.

## Setup

```
pip install -r requirements.txt
```

## Run tests
### Run all tests

```
py.test test
```
